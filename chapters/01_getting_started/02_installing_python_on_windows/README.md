## Practice Scripts

Run these scripts from the project root:

```bash
python3 chapters/01_getting_started/02_installing_python_on_windows/check_python_version.py
python3 chapters/01_getting_started/02_installing_python_on_windows/show_python_path.py
python3 chapters/01_getting_started/02_installing_python_on_windows/check_command_path.py
python3 chapters/01_getting_started/02_installing_python_on_windows/check_pip.py
```

- `check_python_version.py`: confirm which Python version is running.
- `show_python_path.py`: show the Python executable path.
- `check_command_path.py`: check whether Python-related commands are available.
- `check_pip.py`: confirm whether `pip` works from the current Python.

Windows commands to compare:

```powershell
py --version
python --version
py -m pip --version
```
