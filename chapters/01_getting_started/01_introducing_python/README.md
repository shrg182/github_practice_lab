## Practice Scripts

Run these scripts from the project root:

```bash
python3 chapters/01_getting_started/01_introducing_python/hello_python.py
python3 chapters/01_getting_started/01_introducing_python/python_info.py
python3 chapters/01_getting_started/01_introducing_python/basic_calculations.py
python3 chapters/01_getting_started/01_introducing_python/about_python.py
```

- `hello_python.py`: print a simple greeting and short description.
- `python_info.py`: show information about the Python interpreter.
- `basic_calculations.py`: use Python as a calculator.
- `about_python.py`: store simple facts in variables and print them.
