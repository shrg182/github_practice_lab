## Practice Scripts

Run these scripts from the project root:

```bash
python3 chapters/01_getting_started/04_meeting_the_interpreter/interpreter_calculator.py
python3 chapters/01_getting_started/04_meeting_the_interpreter/check_types.py
python3 chapters/01_getting_started/04_meeting_the_interpreter/use_help.py
python3 chapters/01_getting_started/04_meeting_the_interpreter/interactive_prompt_demo.py
```

- `interpreter_calculator.py`: practice arithmetic expressions.
- `check_types.py`: inspect values with `type()`.
- `use_help.py`: explore `help()` and docstrings.
- `interactive_prompt_demo.py`: simulate commands typed at the `>>>` prompt.

Interpreter commands to try:

```pycon
>>> 3 + 4
>>> type("Python")
>>> help(print)
>>> dir(str)
```
